# Complete C# Masterclass — Denis Panjuta

Repositório de estudos do curso **Complete C# Masterclass** do professor [Denis Panjuta](https://www.udemy.com/user/denispanjuta/) na Udemy.

---

## Sobre o Curso

O curso ensina C# do zero ao avançado, com temas como programação orientada a objetos, construção de interfaces gráficas com WPF, desenvolvimento de jogos com Unity 3D, manipulação de dados com LINQ e expressões lambda, além de práticas de Clean Code e testes unitários.

---

## Tecnologias Utilizadas

- **Linguagem:** C# (.NET 10.0)
- **IDE:** Visual Studio 2022
- **Plataforma:** Windows 11
